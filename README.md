# Childress &amp; Childress Moving Co.
### The project is meant as practice working collaboratively with others. In it, we are creating a mock up for a fictional company's brochure site.

Our goal is to explore writing different JavaScript components together to build a fully functional website (albeit fictional). This repository is meant for us to be able to do code reviews on one another, explore concepts together, and see how complex we can build a site. If it progresses well, we'll host it and set up a live link.

Below are a few points of note.

## Components to Incorporate

* Login and Authorization
* RESTful APIs & Database Connections
* Gallery Carousel
* Contact Form w/ email conformation
* CRUD Application
* and others...

## Tech Stack We're Exploring

* React.js/Next.js
* Styled Components or TailwindCSS
* MUI
* MongoDB

## Tools We're Exploring

* TypeScript
* Jest
* Storybook
* Framer Motion
